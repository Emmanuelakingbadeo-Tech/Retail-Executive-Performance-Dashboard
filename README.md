Retail Executive Performance Dashboard

Project Overview

This project was developed as a Data Analytics case study for NexaGen Retail Group, focusing on revenue analysis, profitability, customer segmentation, regional performance, and executive reporting using Power BI.

The dashboard analyzes retail performance across revenue, profitability, customer segments, regional performance, return rates, and quarterly trends.

Business Questions

1. Total Revenue vs Benchmark
2. Gross Profit by Category
3. GP Margin %
4. Monthly Revenue Trend
5. Top Region by Revenue
6. Return Rate by Category
7. Revenue by Customer Segment
8. Quarterly Performance
9. Executive Summary & Recommendations
10. KPI Monitoring Dashboard

Tools Used

* Power BI
* DAX
* Data Modeling
* Data Cleaning
* Business Intelligence

Key Metrics

* Total Revenue: 1.26M
* Gross Profit: 555.5K
* GP Margin: 44%
* Return Rate: 9.5%
* Average Order Value: 6K

Key Insights

* Revenue exceeded expectations and reached 1.26M.
* GP Margin outperformed the 40% industry benchmark.
* Top-performing regions contributed significantly to total revenue.
* Return Rate remained at 9.5%, presenting opportunities for operational improvement.
* Customer segment analysis revealed balanced revenue contributions across segments.
